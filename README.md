# prova-de-nivell

Un teatre especialitzat en musicals té una cartellera amb cuatre obres.
Cada obra consta de:
- Nom
- Duració
- Quantitat de cançons
- Quantitat de personatges
- subgènere )´Clàssic, Dramàtic, Ínfantil o Jukebox)


s'ha poder filtrar les obres per subgènere i buscar el musical més llarg.
